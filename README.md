# CODSOFT — Artificial Intelligence Internship

Task submissions for the **CodSoft Artificial Intelligence Internship** — three complete, working projects covering rule-based NLP, game-playing AI, and content-based recommendation systems.

---

## 📂 Projects

### 🤖 Task 1 — RuleBot: Rule-Based Chatbot
A modular, menu-driven command-line chatbot built entirely with Python's standard library — no AI models or external APIs. Handles general conversation, a built-in calculator, string and number utilities, unit conversion, a Fun Zone (jokes, dice, Rock-Paper-Scissors), a study assistant, and password strength tools.

**Tech:** Python · Standard Library · Modular Programming<br>

🔗 [View Code](https://github.com/anubhav-13579/CODSOFT/tree/main/Task%201%20Chatbot)

### 🎮 Task 2 — Tic-Tac-Toe AI
A modern, animated Tic-Tac-Toe game featuring an unbeatable AI opponent powered by the Minimax algorithm. Supports both AI and local two-player modes, light/dark themes, sound effects, live score tracking, and a fully responsive interface.

**Tech:** React · Vite · JavaScript · HTML5 · CSS3<br>

🌐 [Live Demo](https://anubhav-13579.github.io/Tic-Tac-Toe-/) · <br>🔗 [View Code](https://github.com/anubhav-13579/CODSOFT/tree/main/Task%202%20Tic%20Tac%20Toe)

---

### 🎬 Task 3 — CineMatch: Content-Based Movie Recommendation System
A dark-themed desktop movie recommender inspired by popular streaming platforms. Recommends similar movies using TF-IDF vectorization and Cosine Similarity, with instant title search, Top Rated and Trending sections, persistent favorites, side-by-side movie comparison, genre browsing, and a match-percentage explanation for every recommendation.

**Tech:** Python · CustomTkinter · Pandas · NumPy · Scikit-learn · TF-IDF Vectorizer · Cosine Similarity · Pillow<br>
🌐 [Live Demo](https://github.com/anubhav-13579/CODSOFT/tree/main/Task%204%20Recommendation%20System) *(Desktop App)* ·<br>
🔗 [View Code](https://github.com/anubhav-13579/CODSOFT/tree/main/Task%204%20Recommendation%20System)
