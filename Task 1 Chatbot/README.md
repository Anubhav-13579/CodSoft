# Professional Rule-Based Chatbot in Python

This is a professional, modular, menu-driven rule-based chatbot built entirely using the Python Standard Library. 

It demonstrates clean code practices, modular design, string/number manipulations, interactive menus, and robust error handling without using any external APIs or machine learning frameworks.

## Features

1. **General Conversation**: Responds to greetings and basic questions.
2. **Date & Time**: Displays current date, time, day, month, and year.
3. **Calculator**: Perform basic math operations including percentages and roots.
4. **String Utilities**: Reverse strings, count vowels, check palindromes, etc.
5. **Number Utilities**: Check prime/armstrong numbers, fibonacci, LCM, GCD, etc.
6. **Unit Converter**: Convert lengths, weights, temperatures, and time.
7. **Fun Zone**: Jokes, quotes, facts, and mini-games (Dice roll, Coin toss, Guess the number, Rock Paper Scissors).
8. **Study Assistant**: Ask common tech/computer science questions and get educational answers.
9. **Password Tools**: Evaluate password strength and generate strong passwords.

## Installation

No external dependencies are required! You only need Python 3 installed.

## How to Run

Navigate to the directory and run the main `chatbot.py` script:

```bash
python chatbot.py
```

## Structure

- `chatbot.py`: Main entry point containing the interactive menu and session loop.
- `utils.py`: Reusable utilities for colorful CLI output and user input validation.
- `calculator.py`: Arithmetic functions.
- `string_utils.py`: String manipulation logic.
- `number_utils.py`: Number manipulation and checking.
- `converter.py`: Unit conversion logic.
- `fun_zone.py`: Minigames and randomized content.
- `study.py`: Predefined Q&A mappings.
- `password_tools.py`: Password logic.

Enjoy exploring the chatbot!
